# bad-flappy-bird
![CPP](https://ziadoua.github.io/m3-Markdown-Badges/badges/C++/c++1.svg)

### Compile & run
```
g++ .\main.cpp -I SFML\include\ -L SFML\lib\ -lsfml-main -lsfml-graphics -lsfml-system -lsfml-window -lsfml-audio
./a.exe
```
